# MCP-DAGWO Reproducibility Settings and Parameters

This repository provides the experimental settings and parameter configurations associated with the manuscript on MCP-DAGWO-based harmonic-source localization in IEEE 33-bus and IEEE 69-bus distribution networks.

## Contents

- `parameters/`: algorithm, shrinkage, local support refinement, and compared-method parameter settings.
- `settings/`: benchmark configurations, measurement-node sets, source-node settings, and stress-test case definitions.

## Scope

This repository is intended to document the settings and parameter values used in the reported experiments. Numerical result tables and figures are reported in the manuscript. This repository does not provide a complete software package.

## Notes

All statistical experiments use 30 independent repetitions unless otherwise stated. The parameter values listed here correspond to the reported benchmark settings.
