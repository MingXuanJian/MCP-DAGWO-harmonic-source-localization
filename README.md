# MCP-DAGWO Harmonic Source Localization

This repository provides the implementation code, benchmark scripts, parameter settings, and plotting scripts used in the paper.

## Contents
- IEEE 33-bus experiments
- IEEE 69-bus experiments
- Ablation studies
- Robustness and stress tests
- Plotting scripts

## Requirements
MATLAB R20XX or later

## Usage
Run the main scripts in each experiment folder to reproduce the reported results.
